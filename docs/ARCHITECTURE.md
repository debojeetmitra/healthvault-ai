# HealthVault AI Architecture

## Project Overview

HealthVault AI is a privacy-first healthcare platform built on Midnight Blockchain.

Patients can securely upload medical reports, receive AI-powered explanations, and control doctor access using blockchain-based permissions.

---

## Core Modules

- Authentication
- Patient Dashboard
- Doctor Dashboard
- Medical Report Management
- AI Report Summary
- Blockchain Permission Management
- Audit Logs
